IntelliJ IDEA 提交代码到 GitHub

发表于 2020-05-02

1、环境：

- 系统：Win7
- IDE：IntelliJ IDEA 2018.1.5

2、安装GIT，并在IDEA中绑定

1.下载Git:https://git-scm.com/download/win，（PS：此处我下载的PortableGit，有洁癖不想安装的可以下载使用）
2.设置相关绑定
Settings—>Version Control—>Git（提前下载好）—>Path to Git executable—>选择你提前下载好的的 git.exe 安装目录

3.测试,看到版本说明Git绑定成功

3、配置GitHub，

  1. Settings ——Version Control——GitHub（Host：github.com，Login:账号，Password：密码 ）
